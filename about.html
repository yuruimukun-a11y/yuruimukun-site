yuruimukunについて

yuruimukun（ゆるいむくん）は、日常の隙間に置いておける音楽を少しずつ積み上げていく個人音楽プロジェクトです。このサイトでは、制作したオリジナル楽曲を無料でストリーミング配信しながら、曲の背景や考え方も一緒に残しています。

目指しているのは「ただ曲数が多いサイト」ではなく、作業中、考え事の途中、眠る前に、また戻ってきたくなる音楽IPです。ジャンルは広めですが、入口は絞っています。作業用BGM、睡眠前の静かな曲、情景や物語が立ち上がる幻想寄りの曲。この3つが、ゆるいむくんの軸です。

このIPが目指していること

「作業中、考え事の途中、何もしたくない時間に、そっと流れていることを想定して作っています。」という感覚が、このプロジェクトの根底にあります。主役として前に出すぎないけれど、空気だけは確かに変える。そんな音楽を増やしていくことが、ゆるいむくんの中心です。

押しつけがましくなく、でも確かに何かを感じさせること。日常の中で使えること。タイトルや情景から小さな物語が立ち上がること。こうした要素を、ボカロ曲、和風ヒーリング、ローファイ、エレクトロ、J-Rock、ゲームミュージック風の楽曲にまたがって育てています。

3つの入口

  • 作業用BGM / chill: 集中を邪魔しないけれど無機質になりすぎない、生活の後ろに置ける曲。
  • 睡眠・静けさ / healing: 深夜、休憩、眠る前に小さく流したい静かな曲。
  • 幻想・情景 / story: 夕暮れ、空、街、和の空気感など、景色や物語から入れる曲。

制作について

制作は、DAWでの打ち込み、ギター録音、音作り、配信準備、サイト更新までを一人で積み上げています。曲によってはAIツールや映像生成も使いますが、何を残して何を削るか、どういう気分の曲にするかという判断は、個人制作ならではの手触りとして大事にしています。

すべての楽曲はHLS（HTTP Live Streaming）形式で配信しており、通信環境に応じて最適な品質で再生されます。ダウンロードや会員登録は不要です。

このサイトについて

yuruimukun.comは、制作した楽曲を気軽に聴いてもらうだけでなく、ゆるいむくんの世界観をまとめて蓄積していくための個人運営サイトです。トップページは入口、楽曲ページは作品アーカイブ、ブログは制作背景や考え方を残す場所という役割で運用しています。以下の技術を使って構築しています。

  • フロントエンド：HTML / CSS / JavaScript（バニラ）
  • 音楽配信：HLS形式（Cloudflare R2でホスティング）
  • デプロイ：Vercel
  • 再生数計測：Firebase Realtime Database

チャンネルの役割

  • このサイト: 検索流入、回遊、楽曲説明、世界観の蓄積。
  • YouTube: ShortsやMVで初めて知ってもらう入口。
  • ブログ: 曲の背景、音作り、制作思想などを文章で残す場所。
