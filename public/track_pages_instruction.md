# Claude Code用指示文: 曲説明ページ大量作成

## 目的
AdSense審査を通過するため、各曲に独立した説明ページを作成する。
「有用性の低いコンテンツ」判定を覆すため、**読み応えのあるテキストコンテンツ**を各ページに盛り込む。

### 1. 灯 (akari)
**ジャンル:** アンビエント / エレクトロニカ
**説明:**
暗闘の中でゆらめく小さな灯火をイメージした楽曲。静謐なシンセパッドが空間を包み込み、かすかなノイズが夜の静けさを演出する。忙しい日常から離れ、自分だけの時間に寄り添う音楽として制作をしました。深夜の読書や、眠りにつく前のひとときに最適かも。
**制作背景:**
ある冬の深夜、窓から見えた街灯の光が雪に反射して揺らいでいた。その光景が頭から離れず、「光と闘のグラデーション」を音で表現したいと思ったのがきっかけ。ミニマルな構成でありながら、聴くたびに新しい発見がある音像を目指した。
**聴きどころ:**
1分30秒あたりから現れる微かなベルの音色に注目してくれたら嬉しいです。

### 2. oyasumi
**ジャンル:** チルアウト / ローファイ
**説明:**
「おやすみ」と言いたくなるような、穏やかで温かみのある楽曲。柔らかなピアノとふわりとしたシンセが、一日の終わりを優しく包み込む。激しい展開は一切なく、ただ静かに流れていく時間を音楽にした。眠りへ誘う子守唄のような存在をイメージしました。
**制作背景:**
睡眠用BGMとして聴ける音楽を作りたかった。既存のスリープミュージックは無機質なものが多いと感じていたため、人の温もりを感じられる音色選びにこだわった。録音環境のホワイトノイズをあえて残すことで、生の空気感を演出しています。
**聴きどころ:**
低音域の処理に注目。一般的なローファイとは異なり、重すぎず軽すぎない絶妙なバランスで、ヘッドフォンでもスピーカーでも快適に聴けると思います。

### 3. maron battle
**ジャンル:** エレクトロニック / ゲームミュージック
**説明:**
栗をテーマにした遊び心溢れるバトル曲。ポップでありながらどこか緊張感のあるメロディーが特徴。ゲームのボス戦や、集中作業のBGMとしても使えるアップテンポな仕上がり。可愛さと格好良さが同居する秋を思い出させる一曲。
**制作背景:**
秋のある日、栗を食べながら「栗が猫さんとバトルしたらどんな音楽が流れるだろう」という謎の発想から生まれた曲。真剣になりすぎない、遊びの精神を大切にしている。意外にも作業効率が上がると好評との声も。
**聴きどころ:**
サビ部分のシンセリードが跳ねるようなフレーズを奏でる。アコギの音色とモダンな音色のセッションを楽しんでほしい。

### 4. danmaku
**ジャンル:** シューティングゲーム風 / エレクトロニック
**説明:**
弾幕シューティングゲームの世界観を音楽で表現。無数の弾が飛び交う緊張感と、それを華麗に避ける爽快感をサウンドに落とし込んだ。速いBPMとゲーム風ギターのアルペジオが、シューティングゲームやってる感じになれば幸いです。
**制作背景:**
弾幕ゲームをプレイ中、画面上の弾幕パターンが音楽に見えた瞬間があった。色とりどりの弾が描く軌道を、そのまま音のフレーズに変換するという実験的なアプローチで制作しました。
**聴きどころ:**
中盤のブレイクダウン後、一気に音が増殖していくセクションはこだわっています。まさに弾幕が展開するような音の洪水が感じられたら幸いです。

### 5. 御礼 (orei)
**ジャンル:** 和風アンビエント / インストゥルメンタル
**説明:**
感謝の叫びを音楽に込めた、心温まるメタル絶叫楽曲。メタルサウンドの響きとメタルな打ち込みサウンドが融合し、日本的な情緒を現代的に表現しました。誰かに「ありがとう」を伝えたくなるような、そんな音楽です。
**制作背景:**
普段なかなか言えない感謝の気持ちを、音楽という形で表現したかった。言葉にすると照れくさいことも、メタル風音楽なら素直に伝えられる。普段の自分の納得のいかないストレスが負の連鎖になっていないか今一度自分を見つめなおす機会とした曲です。
**聴きどころ:**
メタル音色ですが、実はそこまでギターは前に出るような弾き方はしない。意外と素直な曲です。ボーカルが一番荒々しいです。後ミクさんにデスボイスをしてほしかったという申し訳なさと好奇心が融合した曲でもあります。

### 6. おわってはじまって (owatte-hajimatte)
**ジャンル:** ボカロ / ポップ
**説明:**
終わりと始まりは表裏一体——そんなメッセージを込めたボカロ曲。何かが終わることへの寂しさと、新しい何かへの期待が交錯する歌詞が胸に響く。前向きになりたい時に聴きたい、背中を押してくれるような一曲。
**制作背景:**
卒業シーズンに作曲。終わりを悲しむのではなく、「終わるから始められる」という前向きなメッセージを届けたかった。ボカロの無機質な声質が、逆に普遍的なメッセージ性を強めている。この世界は常に終わって始まってを小さなことから大きなことまでその繰り返しで出来ていると感じて作った曲です。
**聴きどころ:**
「終わり」から「始まり」へ移行する世界の切り替わりを入れているのは特徴的かなと感じています。

### 7. ねこさんはすごいうた (nekosan-wa-sugoi-uta)
**ジャンル:** ボカロ / コミカル
**説明:**
猫のすごさを歌い上げた、ユーモア溢れる歌。猫のしなやかさ、気まぐれさ、そして絶対的な可愛さを歌詞で表現。聴くだけで幸せになれる、猫への愛が詰まった楽曲になれば幸いです。毎日どうして猫さんはこんなにも可愛く癒されるのか人の理想像ではないだろうか・・・猫さんのにゃーは、初めは無かったそう。人が笑顔になってくれたからにゃーって言うようになったと・・・。可愛すぎないか・・。確かに、マヌルネコさんとか見てるとにゃーって言わない。えっ？・・・。かわいすぎない
  ・・・etc
**制作背景:**
猫動画を見ていたら止まらなくなり、その勢いのまま作曲。猫の魅力を言語化しようとすると意外と難しいことに気づき、改めて猫さんの深さを知った。まだまだビギナーです。そんな思いも込めながら歌いました。
**聴きどころ:**
この曲を聴いて、猫さんの動画や猫さんに興味を持ってくれたら幸いです。可愛いよ猫さん！

### 8. ハロウィンおかたずけ (halloween-okatazuke)
**ジャンル:** ハロウィン / ポップ
**説明:**
ハロウィンが終わった後の、ちょっと寂しくも温かい雰囲気を描いた楽曲。お祭りの後の余韻、飾りを片付ける時間の穏やかさを音楽に。季節もののイベント曲でありながら、普段使いもできる絶妙なバランスで作りました。シンセや打ち込み強め。
**制作背景:**
ハロウィン当日の曲はたくさんあるけれど、「翌日」を描いた曲は少ないことに気づいた。祭りの終わりにこそある、独特の感傷を音楽にしたかった。
**聴きどころ:**
後半にかけてテンポが少し落ち着いていく構成が、お祭りの終わりを表現している。余韻を楽しむように、最後まで聴いてほしい。

### 9. MIKAN NEKOSAN
**ジャンル:** ポップ / エレクトロニック
**説明:**
みかんと猫さんという、日本の冬を象徴する組み合わせをテーマにした楽曲。こたつでまどろむ猫と、その隣に置かれたみかんの温かな光景を音楽で表現。ほっこりとした気持ちになれる、冬にぴったりの一曲。
**制作背景:**
日本の冬の原風景といえば、やはり「こたつ・みかん・猫」。このトリオを音楽で表現できないかという挑戦から生まれた。オレンジ色の温かみを感じるサウンドになっていれば嬉しいです。
**聴きどころ:**
オシャレな一曲に仕上がりました。温かみのある丸い音色が、みかんと猫の柔らかさを表現している。

### 11. ハロウィンマヌルネコ (halloween-manul)
**ジャンル:** ハロウィン / エキゾチック
**説明:**
マヌルネコとハロウィンを組み合わせた、唯一無二のテーマを持つ楽曲。ハロウィンの夜に何を思うのか。エキゾチックなスケールと不思議なハーモニーが織りなす、摩訶不思議な世界観を演出しました。
**制作背景:**
マヌルネコさんのハロウィン衣装が見たいで作った一曲です。マヌルネコさんの独特な格好が、ハロウィンの妖しさと妙にマッチすると感じた。
**聴きどころ:**
西洋的なハロウィンサウンドとの融合、傾向としてメジャーを多用するけいこうにあるので、どうにも明るさが抜けきらないですがそれでも不協和音を入れて不安感をだすようにしたり音の強弱の使い方で、ハロウィン感を演出した一曲です。

### 12. ninja
**ジャンル:** 和風ロック / エレクトロニック
**説明:**
日本の忍者をテーマにした疾走感溢れる楽曲。Bマイナーの和音階とロックサウンドが融合し、夜を駆け抜ける忍者の姿を描く。BPM129という絶妙なテンポが、緊迫感と格好良さを両立出来たら嬉しいなと言う一曲です。
**制作背景:**
海外の「NINJA」イメージではなく、日本人が持つ忍者像を音楽にしたかった。派手なアクションではなく、闘に潜む緊張感を表現することにこだわった。ギターのリフとシンセのレイヤーが、伝統と現代を繋げれるといいなと言う一曲。
**聴きどころ:**
リードギターとシンセが交互にメロディーを奏でる掛け合いが聴きどころ。SAWパッドが空間を埋め、FXが風切り音の役割をして疾走感を演出している一曲です。

### 13. sekaisora
**ジャンル:** J-Rock / オルタナティブ
**キー:** D メジャー
**BPM:** 92.99
**使用楽器:** リードボーカル、バッキングボーカル、エレキギター（クリーン〜軽クランチ）、エレキベース、ドラムセット
**説明:**
世界と空をテーマにした、壮大でありながら繊細なロックナンバー。Dメジャーの明るいコード感と、丁寧に作り込まれたギターワークが特徴。アルペジオからパワーコードまで、できるギタープレイ詰め込んだ一曲です。
**制作背景:**
空を見上げた時の開放感を音楽にしたかった。ギターのクリーントーンから徐々にドライブがかかっていく構成が、王道のバンドサウンドと言った感じも感じられたら幸いです。もともとバンドでやろうと思って作っていた曲でもあったので懐かしさもあります。
**聴きどころ:**
サビでのオクターブ＆ダブルストップのリードギターがバンドっぽくできたいいなと思ってます。付点8分のディレイ（約488ms）が空間に奥行きを与え、プレートリバーブなんかも試してます。

### 14. celtic music nekos
**ジャンル:** ケルティック / ワールドミュージック
**説明:**
ケルト音楽と猫を融合させた、世界観豊かな楽曲。アイルランドの伝統楽器であるティンホイッスルとボドランが、猫たちの舞踏会を奏でる。70.8BPMのゆったりとしたテンポが、古き良き時代への郷愁を誘う。

### 15. sakana
**ジャンル:** ポップロック / インディー
**説明:**
魚をテーマにした、どこか物憂げなポップロック。Eマイナーの哀愁漂うコード感と、水中を漂うようなPADサウンドが特徴。泳ぎ続ける魚の姿に、人生の寓意を重ねた楽曲。
**制作背景:**
水族館で魚を眺めていた時、「この魚たちはどこに向かっているのだろう」と考えた。ただ泳ぎ続けることの意味、その無心さに美しさを感じ、曲にした。
**聴きどころ:**
PADの揺らぎが水の動きを表現している。ギターのアルペジオが差し込む光のようでもあり、水面のきらめきの様な人生の潤いとして手助けになれば幸いです。

### 16. running culture
**ジャンル:** エレクトロニック / トラップ
**説明:**
現代のランニングカルチャーをサウンドで表現。目まぐるしく変わっていく世界、都市を駆け抜けるようなシンセ。フィットネスのBGMとしてはもちろん、通勤通学のモチベーションアップにも最適な一曲かも、心を整えられる手助けになれば幸いです。
**制作背景:**
街中の過行く時間を肌で感じられるような曲を作りたいと思い作りました。イメージとしては映画のプロローグに流れる壮大なbgmというのを試しに作ってみたいと思った一曲になります。
**聴きどころ:**
壮大なサウンドを使っていて前向きになれるような展開を意識して作りましたので前向きになってもらえたら嬉しいです。

### 17. revolutionary event
**ジャンル:** エピック / シネマティック
**説明:**
革命的な出来事の到来を告げる、壮大なシネマティック楽曲。オーケストラ的なアレンジと、力強いビートが融合。何か大きなことが始まる予感、歴史が動く瞬間の興奮を音楽で描きたかったです。
**制作背景:**
歴史の転換点には必ず音楽があった。人々を突き動かす力、時代を変える熱量を音楽で表現したかった。聴く人の心に火をつけるような、そんな曲を目指した。
**聴きどころ:**
後半のオーケストラ風の構成とシンセベースの重なりで壮大な悲しみの中にある前を向く勇気になるような寄り添える一曲を目指しました。気持ちを切り替える手助けになれば幸いです。

### 18. reverth going back

**制作背景:**
リバーブのテールを逆再生させると、まるで時間が巻き戻るような時間の渦を感じてもらえたら嬉しいです。
**聴きどころ:**

### 19. 踊ろ― (odoro)
**ジャンル:** ダンス / ポップ
**説明:**
「踊ろう」という直球なメッセージを込めたダンスナンバー。難しいことは考えず、ただ身体を動かしたくなるようなビートとメロディー。パーティーでも一人の時間でも、心のままに踊れる一曲。
**制作背景:**
踊ることの純粋な喜びを音楽にしたかった。技術や格好を気にせず、ただ音楽に身を任せる——その原初的な快感を表現している。
**聴きどころ:**
サビのシンプルなメロディーが、繰り返されるほどに中毒性を増していく。シンプルさの中に潜むグルーヴを意識しました。突き抜けて明るい曲を目指しました。

## 実装手順

1. `/tracks/` ディレクトリを作成
2. `track-page.css` を作成（個別ページ用スタイル）
3. 各曲のHTMLファイルを生成
4. 曲一覧ページ（`/tracks/index.html`）を作成
5. トップページから曲一覧へのリンクを追加
6. 各ページの動作確認
7. GitHubにプッシュ → Vercelにデプロイ

---

## 注意事項

- 各ページは500〜800文字のテキストコンテンツを含めること
- 「有用性の低いコンテンツ」と判定されないよう、読み応えのある文章を心がける
- 楽曲の埋め込みプレーヤーを必ず設置（既存プレーヤーと連携）
- 関連楽曲へのリンクで回遊性を高める
- フッターにプライバシーポリシー・利用規約へのリンクを設置
